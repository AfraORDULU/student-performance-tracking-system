# Student Performance Tracking System

Bu proje, öğrenci notlarını bir dosyadan okuyarak ortalama ve harf notu
hesaplayan Python tabanlı basit bir performans takip sistemidir.

## Proje Amacı
Dosya okuma, fonksiyon kullanımı ve karar yapıları gibi temel yazılım
mantıklarını pratik bir örnek üzerinden uygulamak.

## Nasıl Çalışır?
Öğrenci bilgileri `students.txt` dosyasından okunur
Vize, final ve ödev notlarına göre ortalama hesaplanır
Ortalama değere göre harf notu atanır
Öğrencinin geçme veya kalma durumu belirlenir

## Kullanılan Teknolojiler
Python
Dosya okuma/yazma
Temel kontrol yapıları (if-else)
